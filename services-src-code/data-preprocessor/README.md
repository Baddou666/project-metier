# Data Preprocessor Source

Code source Spring Boot du service `data-preprocessor`.

## Prerequis

- JDK 21

## Build

Sous Windows :

```sh
./mvnw clean package
```

Sous Linux ou Git Bash :

```sh
chmod +x mvnw
./mvnw clean package
```

## Test

```sh
./mvnw test
```

## Note

Ce service n'est pas documente ici avec une stack Docker complete dans le repo. Utiliser ce dossier surtout pour compiler et tester le code source.
